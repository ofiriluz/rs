# Assets

שים כאן תמונות אישיות (מומלץ) כדי לתת לאתר טאץ׳ אמיתי.

קבצים שהאתר מצפה להם כברירת מחדל:

- `hero.jpg` — תמונת רקע למסך הפתיחה (אם חסר — משתמש בתמונת רקע אונליין)
- `day1.jpg` — תמונה ליום 1 (יקב, גבינות ויין)
- `day2.jpg` — תמונה ליום 2 (ספא)
- `day3.jpg` — תמונה ליום 3 (תערוכה)

אם הקובץ לא קיים, האתר משתמש אוטומטית בתמונת גיבוי מהאינטרנט (מוגדר ב־`imageFallback` ב־`app.js`). אפשר גם לשנות שמות/נתיבים בתוך `PLAN` ב־`app.js`.
